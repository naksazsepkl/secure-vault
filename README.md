# SecureVault

A decentralized identity management system using Fully Homomorphic Encryption (FHE) on the blockchain.

## Features

- Encrypted identity data storage
- Privacy-preserving access control
- On-chain permission verification
- MetaMask wallet integration
- FHEVM support

## Project Structure

- `backend/`: Hardhat-based smart contracts
- `frontend/`: Next.js React application

## Getting Started

### Prerequisites

- Node.js 18+
- MetaMask browser extension
- FHEVM network access

### Installation

```bash
npm install
```

### Development

Backend:
```bash
cd backend
npm run compile
npm run deploy
```

Frontend:
```bash
cd frontend
npm run dev
```
